Then theoretically you could start a basic HTTP listener by running the following in the command line.


python3 -m http.server 1337

If you want a more useful POST listener, then you can start one in python by writing something like this into a file called listener.py 


from http.server import HTTPServer, BaseHTTPRequestHandler

class Listener(BaseHTTPRequestHandler):
    def do_POST(self):
        length = int(self.headers.get("Content-Length", 0))
        body   = self.rfile.read(length).decode()
        print("Received:\n", body[:200])
        self.send_response(200)
        self.end_headers()
        self.wfile.write(b"ok")

if __name__ == "__main__":
    print("Listener started on port 127.0.0.1:1337")
    print("Enter Ctrl-C to kill process")
    HTTPServer(("0.0.0.0", 1337), Listener).serve_forever()

Then if you had listener.py in your working directory you could run 


python3 listener.py

Alternatively, you could start a python3 shell by typing python3 into a Linux terminal and then paste the above codeto run it
